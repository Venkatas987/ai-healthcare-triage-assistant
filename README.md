# 🏥 AI Healthcare Triage Assistant

An intelligent symptom checker that predicts potential diseases and recommends appropriate specialists using machine learning.

## 📋 Features

- **Multi-disease prediction** (41 diseases covered)
- **Confidence scoring** with probability estimates
- **Specialist recommendations** based on diagnosis
- **Severity assessment** (Low/Medium/High/Critical)
- **Urgency level** guidance
- **Clean, responsive UI** with modern design
- **Real-time symptom search** with autocomplete
- **Feedback collection** for model improvement
